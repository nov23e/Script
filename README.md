- Cyril <i@lia.im>

Usage:

| 1        | wget -N --no-check-certificate https://raw.githubusercontent.com/ikym/script/master/shadowsocksR.sh && chmod +x shadowsocksR.sh && bash shadowsocksR.sh       |
